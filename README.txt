Maak een blog in PHP die aan de volgende eisen voldoet:

	· Dient om Taken bij te houden, en is dus een Takenlijst
	· Een overzicht van bestaande Taken kan getoond worden
	· Nieuwe Taken kunnen toegevoegd worden
	· Bestaande Taken kunnen bewerkt worden
	· Bestaande Taken kunnen verwijderd worden
	· Een Taak bestaat tenminste uit: een Titel, Inhoud en Datum
	· De Layout (view) moet minimaal zijn, dus zo eenvoudig mogelijk
	· De code moet minimaal zijn, dus zo eenvoudig mogelijk, maar wel leesbaar en logisch
	· De applicatie maakt gebruik van een database
	· Ook de SQL code moet eenvoudig zijn, maar wel leesbaar en logisch
	· De code moet makkelijk uit te breiden zijn, met nieuwe functionaliteiten

Er moet verschillende versie gemaakt worden:

	1. Een versie in PHP zonder classes en zonder PDO
	2. Een versie in PHP zonder classes en met PDO
	3. Een versie in PHP met classes
	4. Een versie in Laravel, je mag kiezen tussen versie 5.2, 5.4 of 5.6

Al deze versies moeten op elkaar gebaseerd zijn. Je maakt deze opdracht met 2 personen. 
Beide personen moeten de code delen via GIT. En beide personen moeten de code kunnen uitleggen.
